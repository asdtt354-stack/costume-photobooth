# EAIM Costume Studio Booth Edition v3.3

## 촬영 버튼 최종 수정

촬영 버튼이 눌리지 않았던 실제 원인은 `autoBtn`, `scale`, `rotate`,
`saveBtn` 등의 요소를 JavaScript 변수로 선언하지 않고 사용한 것이었습니다.

브라우저에 따라 HTML 요소의 id가 자동 전역 변수로 만들어지지 않기 때문에
모듈 스크립트가 중간에서 ReferenceError로 멈췄고,
그 아래에 있던 촬영 버튼 이벤트도 등록되지 않았습니다.

## 이번 수정
- 모든 버튼과 슬라이더를 `document.querySelector()`로 명시적으로 선언
- 사진 촬영 버튼 이벤트를 명시적으로 등록
- 중복 클릭 이벤트 제거
- 모든 조작 버튼에 `type="button"` 지정
- 얼굴 인식 CDN 로딩 실패와 관계없이 촬영 버튼 작동
- 기본 촬영 모드: 담당자용 후면 카메라
- 최초 접속 관리자 설정 및 Padlet 연결 기능 유지
