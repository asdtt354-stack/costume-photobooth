# EAIM Costume Studio Booth Edition v3.2

## 수정 사항
- 사진 촬영 버튼이 눌리지 않던 문제 해결
- 원인: MediaPipe CDN 정적 import가 실패하면 전체 JavaScript가 중단되어 버튼 이벤트가 등록되지 않았음
- 해결:
  - 얼굴 인식 라이브러리를 동적 import 방식으로 변경
  - 얼굴 인식 로딩에 실패해도 촬영 버튼과 카메라는 작동
  - 사진 촬영 버튼에 별도 안전 이벤트를 추가
- 앱 최초 접속 시 Padlet API Key, Board ID 설정창 자동 표시
- 기본 촬영은 담당자용 후면 카메라

## 확인 방법
1. 새 버전을 GitHub에 덮어씁니다.
2. Vercel 자동 배포 완료를 기다립니다.
3. 휴대전화 브라우저에서 새로고침합니다.
4. 오래된 캐시가 남으면 브라우저 새 탭 또는 시크릿 모드로 접속합니다.
