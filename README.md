# EAIM Costume Studio — Booth Edition

수업나눔 부스 운영을 위해 용고 하나만 남긴 단순화 버전입니다.

## 사용 흐름
1. `체험 시작`
2. 담당자 촬영 또는 셀카 촬영 선택
3. 사진 촬영
4. 용고 얼굴 기준점으로 자동 합성
5. `내 사진 저장`
6. `다음 참가자`

## 특징
- 악기 선택 화면 제거
- 용고 코스튬 1개만 사용
- 후면 카메라 담당자 촬영이 기본
- 셀카 촬영도 선택 가능
- 용고 전용 얼굴 위치·크기·기울기 자동 보정
- 행사 진행자가 빠르게 반복 사용할 수 있도록 단순화

## 설치
압축을 풀고 모든 파일을 GitHub 저장소 최상위에 올린 뒤
Settings → Pages에서 배포하세요.

카메라는 HTTPS에서만 정상 작동합니다.
