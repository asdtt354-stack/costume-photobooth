# EAIM Costume Studio Booth Edition v3.1

## 수정 사항
- 사진 촬영 버튼이 눌리지 않던 JavaScript 실행 순서 오류 수정
- 기본 촬영 모드는 후면 카메라 담당자 촬영
- 앱을 처음 열면 관리자 설정창이 자동으로 열림
- 처음 설정창에서 다음을 입력
  - Padlet API Key
  - Padlet Board ID
  - 참가자용 Padlet 주소
  - 자동 업로드 ON/OFF
- 설정을 저장하면 다음 접속부터 자동으로 불러옴
- 번호 초기화 기능 유지
- 얼굴 기준점은 이전 버전보다 조금 아래로 조정

## 중요
API Key는 앱 코드에 하드코딩되지 않지만, 브라우저 localStorage에 저장됩니다.
따라서 행사 전용으로 관리하는 기기에서만 사용하고, 공용 기기에는 남기지 마세요.
보안상 가장 안전한 방식은 Vercel 환경변수 사용입니다.

## 사용
1. 앱 최초 접속
2. 관리자 설정 자동 표시
3. API Key와 Board ID 저장
4. 사진 촬영
5. 자동 얼굴 합성
6. Padlet 자동 업로드
7. 대취타체험1, 대취타체험2 순서로 번호 증가
